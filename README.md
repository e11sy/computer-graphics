#### Представленный список приложений
1. __Circles and inner tangents__

(10 Вариант)
Приложение предоставляет возможность создавать несколько окружностей и строить внутренние касательные одновременно к двум окружностям. Окружности можно перемещать и менять их радиус с помощью курсора.

2. __B-spline__

(9 Вариант)
Приложение предоставляет возможность задавать B-сплайны различной степени (от 1 до 6) на основе 7 не повторяющихся точек. Точки можно перемещать по плоскости, а также менять степень сплайна.

3. __Bilinear surface__

(1 Вариант)
Построение билинейной поверхности по 4 угловым точкам.
Поверхность отображается в трехмерном пространстве и поворачивается относительно осей X и Y
