## Bilinear surface

Для запуска этого приложения необходимы:

- `Node.js: ^18`
- `npm` или любой другой пакетный менеджер (рекоменуется `yarn`)

1. Установка зависимостей
```shell
yarn
```

2. Запуск dev-сервера
```shell
yarn dev
```

Автоматически откроется страница браузера с приложением:
